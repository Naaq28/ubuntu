Qu'est ce que Ubuntu ?

Ubuntu est un systeme d'exploitation basé sur debian GNU / linux.

C'est un regroupement de logiciels libres qui forme un tout cohérent modulable et adapté à l'utilisateur.

Qu'est ce qu'un logiciel libre ?

permet la liberté d'exécuter le programme, pour tous les usages.

permet la liberté d’étudier le fonctionnement du programme – ce qui suppose l’accès au code source.

permet la liberté de redistribuer des copies – ce qui comprends la liberté de vendre des copies.

permet la liberté d’améliorer le programme et de publier ces améliorations – ce qui suppose, là encore, l’accès au code source.

Qu'est ce qu'un logiciel non libre ?

Il est souvant payants.

Vous n’êtes pas autorisé à partager le logiciel avec d’autres personnes.

Vous ne savez pas comment il fonctionne, vous n’avez pas accès à son code source : son code source est dit fermé.

Vous ne pouvez pas modifier le logiciel pour l’adapter à vos besoins ou corriger un bug. Vous ne pouvez pas non plus demander à une entreprise de le faire pour vous.

Qu'est ce qu'un système d'exploitation ?

c'est un logiciel qui, pilote les dispositifs matériels et reçoit des instructions de l'utilisateur ou d'autres logiciels.

le système d'exploitation gère les processeurs ainsi que la mémoire. Il fait fonctionner les périphériques (clavier, souris, écran, disque dur,...).
